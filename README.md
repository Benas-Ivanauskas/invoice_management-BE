## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Project setup

```bash
$ npm install
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Run tests

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov

```

## Run migrations

```bash
# Create a new Knex migration file with TypeScript extension
$ knex migrate:make migration_name -x ts

# Run all pending migrations in the development environment
$ knex migrate:latest --env development

# Roll back the last batch of migrations
$ knex migrate:rollback

# Roll back all applied migrations
$ knex migrate:rollback --all

# Run the next pending migration (up)
$ knex migrate:up

# Run a specific migration file (up)
$ knex migrate:up 001_migration_name.js

# Roll back the most recent migration (down)
$ knex migrate:down

# Roll back a specific migration file (down)
$ knex migrate:down 001_migration_name.js

# List all migrations and their status (executed or pending)
$ knex migrate:list

```
